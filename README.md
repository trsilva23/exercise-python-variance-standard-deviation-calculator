# Mean-Variance-Standard-Deviation-Calculator (Calculadora de Média, Variância e Desvio Padrão)


## Sobre
Este é um projeto desenvolvido como parte do curso "Data Analysis with Python" da [freeCodeCamp](www.freecodecamp.org).

O objetivo deste projeto é criar uma função em Python que utiliza a biblioteca NumPy para calcular diversas estatísticas
(média, variância, desvio padrão, máximo, mínimo e soma) a partir de uma lista de 9 números, organizando os resultados em um dicionário estruturado.


## Estrutura
*   Validação de entrada para garantir exatamente 9 números.
*   Uso da biblioteca NumPy para operações matemáticas eficientes.
*   Cálculos estatísticos realizados para linhas (eixo 1), colunas (eixo 0) e para a matriz completa (flattened).
*   Retorno dos resultados em um dicionário Python formatado.
  

## Instrução

1.  Clone este repositório:
    ```bash
    git clone github.com
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd mean-variance-standard-deviation-calculator
    ```
3.  Execute o arquivo `main.py` para ver um exemplo de uso e rodar os testes:
    ```bash
    python3 main.py
    ```

## Arquivos no Projeto

*   `mean_var_std.py`: Contém a função principal `calculate()`.
*   `main.py`: Arquivo de exemplo para testar a função `calculate()`.
*   `test_module.py`: Arquivo com os testes de unidade originais do freeCodeCamp
